sql2growl by knippers

SQL Server 2 Growl notification bridge

A bridge to allow notifications from SQL Server (2005 or higher) to Growl (for Windows). It consists of a few database stored procedures, a Service Broker Queue and a C# NT Service that subscribes itself to receiving the notifications from the queue. It uses the Growl for Windows implementation found here: http://www.growlforwindows.com/gfw/.

Install

Unpack the setup.zip and run setup.exe and follow instructions.
